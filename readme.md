## This is TDD Assignment for Incubyte

#### Steps used for creating application 
 Step1: Added gem file \
 ``
 bundle init
 ``

 This command creates a Gemfile in your project directory.

Step 2: Add RSpec for Testing

``
gem 'rspec'
`` \
Then bundled gems: \
``
bundle install
``

Step 3: Setting Up RSpec

``
rspec --init
``
